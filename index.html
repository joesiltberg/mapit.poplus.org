Solve your app’s administrative boundary problems, easily

If your app or website needs to connect locations with administrative areas, like constituencies or counties, MapIt can save you a lot of time.

MapIt is mature, trusted and widely deployed. And it is available as a hosted API or a self-deployed web service, to suit your needs.

Get started

What is MapIt?

MapIt turns geographic points into areas

So 48.85, 2.35 becomes Paris.

Is all about the APIs

Whether you self-host or use a hosted version, you get the data RESTfully.

Can be self-hosted

Deploy an instance and fill it with whatever boundaries you want.

Is also available hosted

Versions pre-loaded with data exist for various areas.

Examples

MapIt UK

The original MapIt, with UK data from Ordnance Survey, ONS, and more.

MapIt Global

An expansion of MapIt for every point on the globe, using OpenStreetMap data.

MapIt Norway
